# Email Spam Classifier
This project predicts whether an email is spam or not using a long short term memory model
# Pipeline
- Words --> vectors

- Tokenization and lemming using nltk

- Bayes score calculated for text

- Bidirectional lstm

- Mail classified into one of 2 categories

# Tech Stack
- NumPy

- PyTorch

- PyTorch_Lightning

- NLTK

- SciKit
